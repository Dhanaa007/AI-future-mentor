# AI Future Me Mentor 

An AI-powered app that generates personalized career roadmaps using Streamlit and Gemini AI.

## Features
- AI roadmap generator
- PDF download
- Saves reports
- Sidebar navigation

## Tech Stack
- Python
- Streamlit
- Gemini API
- ReportLab

## Run Locally
